2. 페이지 제목은 학습지 제작 도우미
3. 기능적인 구현으로 먼저 pdf 파일을 첨부하면 pdf.js 스크립트를 사용하여 해당 pdf 파일의 텍스트를 추출
4. 사용자가 원하는 학습지 형태를 선택( 선택할 수 있는 학습지 형태는 우리가 구조화 - 일단 1번으로 빈칸이 뚫린 학습지 2번으로 형성평가를 만들어주는 학습지)
5. 그리고 부가적으로 사용자가 원하는 프롬프트 입력
6. 그리고 출력할 형태 지정(일단 우선적으로 docs 형태로 추출)
7. GPT API 를 호출하여 위 데이터를 토대로 학습지를 만들어주고
8. 사용자가 사용할수있게 또는 다운받을 수 있게
